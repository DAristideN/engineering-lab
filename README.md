# engineering-lab

Lab personnel (apprentissage + projets + expérimentation) : Python, PowerShell, Data, Agents IA.

## Structure
- `foundations/` : exercices fondamentaux
- `projects/` : mini-projets (CLI, automation)
- `ai-lab/` : agents, RAG, expériences
- `notebooks/` : Kaggle / Coursera (avec notes)
- `docs/` : architecture, décisions, journal d’apprentissage

## Règles
- Ne jamais committer `.env` ni des clés API.
- `data/output/` contient des résultats générés (non versionnés).
- Commits petits et explicites.
